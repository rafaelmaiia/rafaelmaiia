# Hi, I'm Rafael Maia 👋😁

Desenvolvedor Full-Stack com foco em Java, Spring, React e TypeScript.

Atuo na construção de aplicações web, APIs REST e soluções digitais voltadas a problemas de operação e negócio. Minha experiência combina desenvolvimento de software, visão prática de produto e vivência com atendimento, vendas e relacionamento com clientes.

Hoje meu principal projeto é o **Mar de Beleza**, um sistema de gestão para salão que concebi, desenvolvi, implantei e continuo evoluindo. O projeto envolve agenda, clientes, profissionais, serviços, pagamentos e registros financeiros.

O repositório principal é privado por envolver uma aplicação em produção, mas a apresentação pública do case estará disponível em breve.

---

## Stack principal

**Backend:** Java, Spring Boot, Spring Security, Spring Data JPA, APIs REST, JWT, Flyway  
**Frontend:** React, TypeScript, JavaScript, Vite, TailwindCSS  
**Banco de Dados:** PostgreSQL, MySQL, H2  
**Testes:** JUnit 5, Mockito  
**Ferramentas:** Docker, Git, GitHub, Vercel, Railway, Postman

## Case principal

**[Mar de Beleza — Sistema de Gestão para Salão de Beleza](https://github.com/rafaelmaiia/case-mar-de-beleza)**
